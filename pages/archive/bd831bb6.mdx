## Introducción
Mall bot es un bot escrito en python que te permite crear tu propio bar en discord.
Es un bot sencillo de usar y administrar y con el que puedes echarte unas risas con tus amigos.

## Como usarlo

### Comandos de usuario

```bash  filename="Pide un producto al camarero" copy
$order <name>
```


```bash  filename="Pide la carta al camamerero" copy
$carta
```
### Comandos de administración


```bash  filename="Añade un producto a la carta" copy
$insert <name> <category> <url:opt>
```
```bash  filename="Edita el nombre de un producto" copy
$update <old:name> <new:name>
```
```bash  filename="Edita la categoría de un producto" copy
$update <name> <new:category>
```
```bash  filename="Borra un producto" copy
$delete <id>
```
```bash  filename="Borra la imagen de un producto" copy
$clear <name>
```
```bash  filename="Muestra los ids de las imagenes de un producto" copy
$select <name>
```
