## Introducción
El proyecto maps Scraping es un script que permite la extracción de información 
de google maps para su posterior análisis y consulta. Surge de la necesidad de 
tener una base de datos de las floristerías de toda España para poder realizar
una comunidad entre la que poder redirigirse los pedidos de flores entre ellas.

## Como funciona
Para ejecutar el script es necesario tener instalado python 3 instalado en el
sistema. Una vez instalado, se debe ejecutar el script con el siguiente comando:

```bash
python3 maps_scraping.py
```

Una vez ejecutado el script, se le pedirá al usuario que introduzca una query 
sobre la que extraer datos de google maps por ejemplo "floristerias huesca".

El script se encargará de extraer los datos de las páginas que aparecen en la
primera página de resultados de google maps. Para cada resultado se extraerá
la siguiente información:

| Campo | Descripción | Tipo |
| --- | --- | --- |
| title | Título de la página | string |
| score | Puntuación de la página | float |
| num_reviews | Número de reseñas de la página | int |
| tag | Etiquetas de la página | list |
| address | Dirección de la página | string | 
| coords | Coordenadas de la página | tuple |
| domain | Dominio de la página | string |
| phone | Teléfono de la página | string |
| plus_code | Código plus de la página | string |
| email | Email de la página | string |
